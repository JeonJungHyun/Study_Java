# 📗 Study Java (2025.08 ~ 2025.10)

자바 기초 문법과 주요 주제를 공부하며 작성한 코드 기록입니다.

수업 시간에 배운 내용과 복습을 기반으로 개인적으로 다시 실습해봤던 코드들을 이 저장소에 정리했습니다.

처음 공부하던 당시에는 Git 사용법을 몰라 로컬 환경에서만 관리했습니다.

이후 Git을 학습하면서 과거 공부 내용을 정리하여 최근에 업로드하였습니다.


---

## 📚 학습 내용

이 저장소는 아래 주제들을 중심으로 정리되어 있습니다:

### • java01 ~ java05  
- 자바 기본 문법과 실행 구조  
- 기본 입출력과 연산자

### • java06 ~ java10  
- 조건문 (if, switch)  
- 반복문 (for, while)  
- 배열  
- 메서드와 클래스 실습  
- 상속, 다형성

### • java11 ~ java15  
- 추상화, 인터페이스  
- 예외 처리  
- 컬렉션 프레임워크 (List, Set 등)  
- 제네릭 실습

각 디렉터리 안에는 해당 주제별 예제 코드(.java 파일)가 포함되어 있습니다.
